In this lecture we will basically style the lists.
Now first tag is list-style-type: ; this will change the marker type of the List for ex. disc,square,roman etc.
Now lets assume that we have set a background color to the list items and we want that the marker should be visible on the background color then we will use
list-style-position: inside; outside will imply that the marker will not be covered in the background color
Next we have list-style-image:url();.This is used to set an image as a marker.
